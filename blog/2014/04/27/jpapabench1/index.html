[Paparazzi Analysis]: Getting Started With Jpapabench 1

Apr 27th, 2014

I am currently working on a project that related to Paparazzi. Surprisingly, There are only few material available online. I decide to document all my exploration here. most of the things I am writing here is mainly from my code analysis, Paparazzi wiki page and its is related Java version project Jpapabench.

Jpapabench is a Java version of Paparazzi to provide a realistic application benchmark for evaluation variouse Java Virtual Machine (JVM), especilly for Real-time JVM. Since the code base of Paparazzi have been grown dramaticly to adapt to more wild hardware driver support, it is more easier to start with Jpapbench, it mirrors the function of Paparazzi, change the cyclic execution into tasks and execute them in seperated threads. Therefore, I decide to start with Jpapabench first.
